High End Visualization
HEVx

High End Visualization (HEV) is a software environment for developing visualization applications in both desktop and immersive environments.

This development environment is mainly designed for supporting the NIST CAVE and other immersive environments at NIST for visualization purposes. The code is being made publically available for the benefit of the visualization community.

Currently this code is under active development and in a pre-release state.

Building

Requirements

  • CMake 3.11
  • Python 3
    • Wheezy Template: '''pip install –user wheeze.template'''
  • GCC >= 7

Dependencies

The following packages are fetched and managed with CMake:

  • abseil/apseil-cpp
  • Microsoft/GSL
  • g-truc/glm
  • taocpp/json
  • TartanLlama/expected
  • gabime/spdlog
  • 01org/tbb
  • mosra/flextgl

Instructions

On HEV hosts:

$ mkdir build && cd build
$ scl enable devtoolset-7 rh-python36 -- cmake3 -DCMAKE_BUILD_TYPE=RelWithDebInfo ..
$ make -j
